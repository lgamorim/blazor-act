# blazor-act

This app was built for a presentation in a tech community, to showcase the most relevant features of Blazor at the time. It's a snapshot from that talk, not an actively maintained project.

## Running it

**Prerequisites:** [.NET SDK](https://dotnet.microsoft.com/download)

```bash
git clone https://github.com/lgamorim/blazor-act.git
cd blazor-act
dotnet run --project src/WebAssemblyApp
```
